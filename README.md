# 📌 Project Overview

Customer churn is a critical issue for subscription-based businesses. This project builds a **machine learning model to predict whether a customer will leave the service**.

By identifying customers who are likely to churn, businesses can take **proactive retention strategies** to reduce revenue loss.

---

# 🎯 Objectives

✔ Analyze customer behavior patterns  
✔ Identify factors contributing to churn  
✔ Train machine learning classification models  
✔ Predict customers who are likely to churn  

---

# 🗂️ Dataset

The dataset contains customer information such as:

- Gender  
- Tenure  
- Monthly Charges  
- Total Charges  
- Contract Type  
- Payment Method  
- Internet Service
- ....etc 22 features  

**Target Variable**

`Churn` → Yes / No

# 🔍 Exploratory Data Analysis

Some important insights discovered:

📊 Customers with **month-to-month contracts churn more frequently**

📊 Customers with **higher monthly charges tend to churn more**

📊 Customers with **longer tenure are more loyal**



# 🤖 Machine Learning Models Used

- Logistic Regression


Models were evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
